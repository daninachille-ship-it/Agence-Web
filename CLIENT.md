# Guide propriétaire — Maison Blend

Bienvenue ! Ce guide vous explique comment modifier le contenu de votre site
en quelques clics, sans aucune connaissance technique.

---

## Accéder à l'interface d'administration

1. Ouvrez votre navigateur et rendez-vous sur :
   **https://votre-site.vercel.app/admin**

2. Connectez-vous avec votre adresse email (compte Tina Cloud)

3. Vous arrivez sur l'interface d'administration

---

## Modifier les informations du café

### Nom, slogan, description

1. Dans le menu de gauche, cliquez sur **"Configuration du site"**
2. Cliquez sur **"Informations générales"**
3. Modifiez les champs : nom, slogan, descriptions, etc.
4. Cliquez sur **"Enregistrer"** en haut à droite

---

## Modifier les horaires

1. Dans **"Configuration du site"**, cliquez sur **"Horaires"**
2. Cliquez sur le jour à modifier
3. Activez/désactivez le toggle **"Ouvert"**
4. Changez les heures d'ouverture et de fermeture
5. Cliquez sur **"Enregistrer"**

---

## Modifier la carte (menu)

### Changer le prix d'un plat

1. Dans **"Configuration du site"**, cliquez sur **"La carte"**
2. Cliquez sur la catégorie (ex: Cafés, Brunch…)
3. Cliquez sur l'item à modifier
4. Changez le prix dans le champ **"Prix"**
5. Cliquez sur **"Enregistrer"**

### Ajouter un nouveau plat

1. Dans la catégorie voulue, cliquez sur **"+ Ajouter"**
2. Remplissez le nom, la description et le prix
3. Cliquez sur **"Enregistrer"**

### Supprimer un plat

1. Cliquez sur l'item à supprimer
2. Cliquez sur l'icône poubelle
3. Confirmez la suppression
4. Cliquez sur **"Enregistrer"**

---

## Modifier les avis clients

1. Dans **"Configuration du site"**, cliquez sur **"Avis clients"**
2. Cliquez sur un avis pour le modifier, ou **"+ Ajouter"** pour en créer un
3. Remplissez : texte de l'avis, nom de l'auteur, date
4. Cliquez sur **"Enregistrer"**

---

## Changer les photos

1. Dans **"Configuration du site"**, cliquez sur **"Photos"**
2. Cliquez sur l'emplacement à changer (ex: "Photo principale (Hero)")
3. Cliquez sur **"Choisir un fichier"** et sélectionnez votre photo
4. Attendez que l'upload soit terminé
5. Cliquez sur **"Enregistrer"**

**Formats acceptés :** JPG, PNG, WebP
**Conseil :** utilisez des photos de bonne qualité (min. 1000px de large)

---

## Modifier les informations de contact

1. Dans **"Configuration du site"**, cliquez sur **"Contact & Localisation"**
2. Modifiez l'adresse, le téléphone, l'email, Instagram, etc.
3. Cliquez sur **"Enregistrer"**

---

## Questions fréquentes

**Les modifications sont-elles immédiates ?**
Après avoir cliqué "Enregistrer", les modifications sont publiées en
quelques secondes. Rechargez la page de votre site pour les voir.

**J'ai fait une erreur, comment annuler ?**
L'interface conserve un historique des modifications. Contactez votre
agence web si vous avez besoin de restaurer une ancienne version.

**Le site est lent à charger ?**
Les photos trop lourdes peuvent ralentir le site. Essayez de compresser
vos images avant de les uploader (outil gratuit : squoosh.app).

---

## Besoin d'aide ?

Contactez votre agence web pour toute question ou modification plus avancée.
